# Game of Life
This is my implementation of Conway's Game of Life in C++ using Raylib.

## Features
- [x] Toggle Cells
- [x] Paint Cells
- [x] Play/Pause
- [x] Infinite Grid
- [x] Step
- [x] Zoomable
- [x] Adjustable Step Speed for Play/Pause
- [x] Fun
